# Spell Checker with Binary Search Tree

A spell checker implementation using a Binary Search Tree data structure to store and search dictionary words.

## Features
- Loads dictionary from text file
- Checks input text for misspelled words
- Interactive word addition to dictionary
- O(log n) search and insertion

## Files
- `BST.java` - Generic Binary Search Tree implementation
- `Dictionary.java` - Interface defining dictionary operations
- `SpellChecker.java` - Spell checking functionality
- `Main.java` - Interactive demo program
- `dictionary.txt` - Base dictionary file for test usage

## Usage
1. Compile: `javac *.java`
2. Run: `java Main`
3. Enter text to check for spelling errors
4. Optionally add misspelled words to dictionary

## Requirements
- Java 21 or higher
- Dictionary file in same directory as compiled classes
